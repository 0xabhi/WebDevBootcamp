# NodeJs-
### require dependency-
```
var dep = require('dependency");
```

### Npm install dependency-
```
npm install dep
```

### Express-
```
var app=require("express");

res.send("");
});
```

```
//Tell express to listen
app.listen(3000,function(){
});
```

npm init
